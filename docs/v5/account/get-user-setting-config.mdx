You can get configuration how the system behaves when your limit order price exceeds the highest bid or lowest ask price.

Spot
* **Maximum Buy Price**: *Min[Max(Index, Index × (1 + y%) + 2-Minute Average Premium), Index × (1 + z%)]*
* **Lowest price for Sell**: *Max[Min(Index, Index × (1 – y%) + 2-Minute Average Premium), Index × (1 – z%)]*


:::info Default Setting
*	Spot:
**lpaSpot = false.** If the order price exceeds the boundary, the system rejects the request.

:::

### HTTP Request
GET `/v5/account/user-setting-config`

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|result |array |Object |
|> lpaSpot |boolean |<ul><li>`true`: If the order price exceeds the boundary, the system will automatically adjust the price to the nearest allowed boundary</li><li>`false`: If the order price exceeds the boundary, the system rejects the request.</li></ul>|
