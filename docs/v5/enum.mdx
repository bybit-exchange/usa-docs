* `GTC`&emsp;GoodTillCancel

* <a href="https://www.abfinance.com/en/help-center/article/Retail-Price-Improvement-RPI-Order">RPI</a>&emsp;features:<ul><li><b>Exclusive Matching</b>: Only match non-algorithmic users; no execution against orders from Open API.</li>
<li><b>Post-Only Mechanism</b>: Act as maker orders, adding liquidity</li>
<li><b>Lower Priority</b>: Execute after non-RPI orders at the same price level.</li>
<li><b>Limited Access</b>: Initially for select market makers across multiple pairs.</li>
<li><b>Order Book Updates</b>: Excluded from API but displayed on the GUI.</li></ul>
