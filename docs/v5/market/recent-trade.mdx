Query recent public trading history.
> **Covers: Spot**

You can download archived historical trades from the [website](https://www.abfinance.com/derivatives/en/history-data)

### HTTP Request
GET `/v5/market/recent-trade`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |**true** |string |Product type. `spot`|
|[symbol](../enum#symbol) |**true**  |string |Symbol name, like `BTCUSDT`, uppercase only|
|limit |false |integer |Limit for data size per page <ul><li>`spot`: [1,60], default: `60`</li></ul>|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|category |string |Products category |
|list |array |Object |
|> execId |string |Execution ID |
|> symbol |string |Symbol name |
|> price |string |Trade price |
|> size |string |Trade size |
|> side |string |Side of taker `Buy`, `Sell`|
|> time |string |Trade time (ms) |
|> isBlockTrade |boolean |Whether the trade is block trade |
|> isRPITrade |boolean |Whether the trade is RPI trade |
|> mP |string |Mark price, unique field for `option`|
|> iP |string |Index price, unique field for `option`|
|> mIv |string |Mark iv, unique field for `option`|
|> iv |string |iv, unique field for `option`|
|> seq |string |cross sequence|
