### HTTP Request
GET `/v5/market/time`

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| timeSecond | string | Server timestamp (sec)  |
| timeNano | string | Server timestamp (nano)  |
