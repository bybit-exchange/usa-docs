獲取服務器時間

:::info
* 在極端市場波動期間, 此介面可能會出現延遲增加或資料傳遞暫時延遲的情況
:::

### HTTP 請求
GET `/v5/market/time`

### 請求參數
無

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
| timeSecond | string | 服務器時間戳 (秒)  |
| timeNano | string | 服務器時間戳 (微秒)  |
